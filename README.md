## 🔄 更新方式

1. 修改对应的 `.yaml` 文件
2. 在 OpenClaw 中用 `cron update` 同步
3. 提交到本仓库

## 📝 导出方式

```bash
# 从 OpenClaw 导出
openclaw cron list --format yaml > cron-configs.yaml
```
